# 🤖 Meu Primeiro Copilot

Esse repositório foi criado como parte do curso da DIO sobre GitHub Copilot.
A proposta é aprender a trabalhar com os diferentes modos de interação com IA pra otimizar o dia a dia de desenvolvimento.

---

```

│
├── prompts/
```

---

## 🧩 Os 5 Modos do Copilot

### ❓ Ask
Use quando quiser **compreender** algo sem alterar o código.
Ótimo pra resolver dúvidas sobre erros, conceitos ou funcionalidades.

### ✏️ Edit
Use quando quiser **alterar** um trecho de código já existente.
Refatoração, conserto de bugs, ganho de performance...

### 🧭 Plan
Use quando a tarefa for **grande ou complicada**.
O copilot organiza tudo antes de começar a codar, então você valida a estratégia primeiro.

### 🤖 Agent
O modo mais **independente** de todos.
Ele percorre o projeto, cria arquivos, modifica vários pontos e conserva o contexto entre cada etapa.

### 📚 Study
Use quando quiser **aprender** de verdade, sem só copiar a resposta.
Funciona como um tutor que orienta o seu raciocínio.

---

## 🚀 Como usar

1. Copie o conteúdo do prompt que quiser utilizar
2. Cole no chat do GitHub Copilot (ou qualquer outra IA)
3. Comece a interagir!
